1) WHAT THE FEEDBACK MEANS (ACTIONABLE INTERPRETATION)
-----------------------------------------------------------------
1. "not a lot of 6mA"
- Your A-mod (Reg proxy) signal is sparse/noisy.
- Do not over-claim biological conclusions from raw A-mod counts.

2. "use original value of M (easy mark only 0 or 1)"
- Prefer binary methylation labels (methylated/unmethylated) instead of transformed continuous M when possible.
- Keep unknown as missing mask, not as numeric value.

3. "normalize reg to 0..1 and correlate with M"
- Convert Reg feature to a bounded scale (per-bin or per-read normalization).
- Then compute correlation/joint clustering against binary or fraction M.

4. "need to understand log1 in contour"
- `log1p(x)` means `log(1+x)`.
- It compresses long tails so high-count outliers do not dominate.
- You must state this clearly in figure caption.

5. "use window of 1 kb and not 1 Mb; cut genome more intelligently"
- Keep 1 Mb as context container only if needed for model context.
- For analysis/visualization, compute features on 1 kb bins.
- Prefer filtering to informative bins (coverage + variance/homogeneity criteria).

6. "contour plots, add histograms"
- Use joint plot style: center = scatter/KDE contours, top/right = marginal distributions.

7. "on contour plot do topographic maps"
- Use density contour levels like elevation lines (KDE contour map).

8. "legend each sample/mark"
- Overlay marks in one panel with clear legend and distinct colors.
- Ensure h3k27ac is visually separated from me3 marks.

3) PHASE 2 REVISED ANALYSIS STRATEGY
-----------------------------------------------------------------
Goal
- Keep long-context compatibility for modeling, but perform statistical analysis at 1 kb scale.

3A) Bin size and filtering
- Use 1 kb bins for O/E and joint distribution analyses.
- Keep only bins with sufficient C coverage.
- Add optional "informative bin" filter (exclude near-constant bins).

3B) Segmentation refinement ("more intelligent cuts")
- Instead of treating every 1 Mb equally, annotate bins/segments by:
  - local variance of M/Reg
  - coverage stability
  - optional genomic annotations (later: peaks, chromatin state)
- Analyze homogeneous segments separately from heterogeneous segments.

3C) Keep 1 Mb for model context, 1 kb for features
- This resolves the apparent contradiction:
  - model can still ingest long context windows
  - statistical plots use biologically/local meaningful 1 kb bins
