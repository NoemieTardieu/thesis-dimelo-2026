# Enhanced Phase 1 Visualizations

Implemented upgrades:
- Upgrade A: log2 observed/expected enrichment heatmaps (independence baseline).
- Upgrade B: joint contour maps on log1p(Reg density), log1p(M density).

Interpretation of Upgrade A:
- 0 (white): independence between Reg and M bins.
- >0 (red): enriched co-occurrence beyond marginals.
- <0 (blue): depleted co-occurrence.

Upgrade C (peak vs non-peak) is not included yet because peak annotations were not provided in this phase.
