# Old vs DNA+5mC HyenaDNA Population Benchmark Comparison

This folder compares the previous HyenaDNA population track with the final DNA+5mC-conditioned HyenaDNA track.

- `hyenadna_model_metric_comparison.tsv`: old/new pairwise metrics involving HyenaDNA.
- `hyenadna_model_metric_deltas.tsv`: new-minus-old metric deltas.
- `canonical_raw_and_global_normalized_old_new_hyena.tsv.gz`: merged bin table with both HyenaDNA tracks.
- `overlay_selected_regions_old_new_hyena/`: locus plots with both HyenaDNA lines.
